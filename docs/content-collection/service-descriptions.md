# Content Collection — Service Descriptions

**Goal:** confirm or refine the description for each photography category. Drafts already exist in `content/services.ts` in the documentary brand voice — this template gathers what's needed to make each one truly yours.

> Five services: **Famille · Couple · Grossesse · Mariage · Portrait.** Each links to its gallery (slugs: `familles`, `couples`, `grossesse`, `mariages`, `portraits`).

---

## For EACH service, please confirm/provide

1. **Tagline** — one emotional line (the feeling, not the feature).
   *Draft example (Couple): "La façon dont vous êtes ensemble."*
2. **Description** — 1–2 short paragraphs: what this kind of session is really like with you.
3. **Approach** — 2–3 short bullets: how it actually runs (location, light, posing, color/B&W).
4. **Ideal for** — who books it / typical occasions.
5. **Anything specific** — e.g. maternity from which month; wedding half-day vs full-day; portrait for personal vs professional.

---

## Per-service prompts

### Famille
- Where do family sessions usually happen (home / outdoors)?
- How do you handle young or shy children?
- What makes a family session with you different?

### Couple
- Outdoor / golden hour? How much direction vs letting them be?
- Do you offer this as a "before the wedding" warm-up?

### Grossesse
- From which month do you recommend?
- How do you keep it comfortable and tasteful?
- Solo / with partner / with older children?

### Mariage
- Half-day vs full-day? What's included?
- Reportage-only or some posed couple portraits?
- Do you travel for weddings? Second shooter ever?

### Portrait
- Personal, artistic, professional — all three?
- Studio / home / outdoor?
- Color or B&W lead?
